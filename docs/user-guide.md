## 1. 開始使用

Live English Tutor 是一款由 AI 驅動的即時英語口說練習平台。你將與 AI 家教 **Emma** 進行真實語音對話，Emma 會在對話過程中即時糾正你的英文，並在課程結束後提供個人化的中文學習報告。

**你需要準備：**

- 一台有麥克風的電腦或手機
- Google 帳號（用於登入）
- 穩定的網路連線

---

## 2. 登入帳號

### 步驟

1. 打開瀏覽器，前往應用程式網址
2. 點擊 **「Continue with Google」** 按鈕

   ![登入頁面](./assets/login.png)
   > *畫面會出現 Google 授權視窗，選擇你要使用的 Google 帳號。*

3. 授權完成後，系統會自動跳轉到課程選擇頁面（Dashboard）

> **注意：** 本平台目前僅支援 Google 帳號登入。第一次登入會自動建立你的帳戶，無需另外註冊。

### 登出

點擊右上角的 **「Sign out」** 即可安全登出。

---

## 3. 課程主題選擇

登入後，你會看到以下五個主題卡片，點擊任一主題即可開始課程：

| 主題 | 適合練習內容 |
|------|-------------|
| 💬 **General Conversation** | 日常聊天、自由對話、輕鬆暖身 |
| 💼 **Job Interview English** | 面試自我介紹、STAR 法則回答技巧 |
| ✈️ **Travel English** | 機場、飯店、餐廳等旅遊情境用語 |
| 📈 **Business English** | 會議、電郵、簡報等商務英文 |
| 🏠 **Daily Life English** | 購物、辦事、日常生活英文 |

點擊後，卡片上會出現 **「Starting…」** 載入提示，系統正在為你建立課程房間並連線到 Emma。

---

## 4. 進行對話課程

### 4-1 課程介面說明

連線成功後，你會進入課程頁面，畫面如下：

```
┌─────────────────────────────────────────┐
│ 🎓 Emma                      [End Lesson]│
├─────────────────────────────────────────┤
│                                         │
│              🎓 Emma 頭像               │
│         (說話時會出現擴散光圈)            │
│                                         │
│          Emma is speaking…              │
│                                         │
│               🎙️ 麥克風按鈕              │
│            Tap to speak / mute          │
│                                         │
├─────────────────────────────────────────┤
│ ✏️ Corrections              [1]          │
│ your → you're                           │
│ 使用縮寫 you're 比 your 更正確           │
└─────────────────────────────────────────┘
```

### 4-2 麥克風操作

- **按下 🎙️ 麥克風按鈕** → 啟動麥克風，開始說話
  - 按鈕變紅、出現脈動光圈代表「正在錄音」
- **再次點擊** → 靜音（Mute）
  - 按鈕變灰、顯示 🔇 代表「已靜音」

> **建議：** 說話時保持穩定音量，避免背景噪音。Emma 會等你說完句子再回應。

### 4-3 課程流程

一堂課程分為三個階段，Emma 會自動引導你：

| 階段 | 說明 | 大約回合數 |
|------|------|-----------|
| **Warm-up** | 輕鬆暖身聊天，Emma 會先自我介紹 | 約 3 回合 |
| **Practice** | 針對主題進行深度對話練習 | 約 10 回合 |
| **Summary** | Emma 總結今日練習重點 | 1 回合 |

### 4-4 即時語法糾正

當你說出的英文有語法或用詞錯誤時，畫面下方的 **Corrections 卡片** 會即時顯示：

- 🔴 ~~錯誤的說法~~
- 🟢 **正確的說法**
- 📝 錯誤說明（中文）

Emma 在對話中也會自然地幫你複習正確說法，不會打斷對話節奏。

### 4-5 結束課程

點擊右上角的 **「End Lesson」** 按鈕，Emma 會整理課程資料並自動跳轉到學習報告頁面。

> 如果你中途關閉瀏覽器或網路斷線，課程也會自動結束並生成報告。

## 5. 課後學習報告

課程結束後，系統會進入報告頁面。Emma 會根據你今日的對話，用**繁體中文**撰寫個人化學習報告。

### 報告生成時間

報告通常需要 **30 秒至 1 分鐘** 生成，頁面會自動更新：

```
📝 Generating your report...
Emma is writing up your feedback.
This usually takes under a minute.
```

### 報告內容結構

報告包含以下五個部分：

1. **課程摘要** — 今日練習主題與整體表現概覽
2. **表現亮點** — 你說得好的地方，給予具體鼓勵
3. **改善建議** — 本次出現的語法錯誤與改進方向
4. **新學單字 / 用語** — 課程中出現的重要詞彙整理
5. **下次練習建議** — Emma 推薦你下次應練習的方向

### 報告完成後

- 點擊 **「Start new lesson →」** 回到主頁選擇新主題
- 或點擊頁首 **「← Dashboard」** 返回課程列表

---

## 6. 歷史紀錄

Dashboard 下方的 **「Recent sessions」** 列表顯示你所有的課程紀錄：

| 欄位 | 說明 |
|------|------|
| 主題 emoji + 名稱 | 當時練習的主題 |
| 開始時間 | 課程建立時間 |
| 狀態標籤 | Pending / Active / Completed |
| View Report → | 僅已完成課程可點擊，查看報告 |

> 所有課程紀錄和報告都會永久保存在你的帳戶中。

## 7. 常見問題

### Q1：Emma 聽不到我說話？

- 確認瀏覽器已獲得麥克風權限（瀏覽器網址列會有提示）
- 確認麥克風按鈕為 **紅色**（代表正在錄音），不是灰色
- 嘗試重新整理頁面並重新連線

### Q2：課程頁面卡在「Connecting to your lesson…」？

- 確認網路連線穩定
- 嘗試重新整理頁面
- 確認後端服務正常運作（請聯繫管理員）

### Q3：我說了很久，Emma 沒有回應？

目前 STT（語音轉文字）功能尚在整合階段。如果 Emma 長時間沒有回應，可能是語音輸入尚未啟用。請聯繫管理員確認服務狀態。

### Q4：報告一直顯示「Generating…」？

- 報告由 AI 生成，通常在 1 分鐘內完成
- 若超過 5 分鐘仍未完成，請重新整理頁面
- 如問題持續，請聯繫管理員確認 Ollama 服務狀態

### Q5：可以重複查看過去的報告嗎？

可以。在 Dashboard 的「Recent sessions」找到已完成的課程，點擊 **「View Report →」** 即可重新查看。

### Q6：支援哪些瀏覽器？

建議使用以下瀏覽器的最新版本：

- Google Chrome ✅
- Microsoft Edge ✅
- Safari（macOS / iOS）✅
- Firefox ⚠️（部分 WebRTC 功能可能受限）
